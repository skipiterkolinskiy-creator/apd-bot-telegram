# APD Telegram Bot

Бот для Astronom Police Department с пятью разделами:

1. Академия
2. Жалоба на сотрудника
3. Благодарность
4. BACKHOME
5. Вопросы

Все обращения отправляются в закрытый чат руководства. У руководства есть кнопки:

- Принять
- Отклонить
- На рассмотрении

Пользователь автоматически получает уведомление о решении.

## 1. Создание бота

1. Откройте в Telegram `@BotFather`.
2. Отправьте `/newbot`.
3. Задайте имя и username.
4. Скопируйте токен.

Никому не отправляйте токен бота.

## 2. Подготовка чата руководства

1. Создайте закрытую группу.
2. Добавьте туда бота.
3. Выдайте боту право отправлять и изменять сообщения.
4. Узнайте ID группы через любого бота для определения ID либо временно добавьте в код вывод входящих обновлений.
5. ID супергруппы обычно начинается с `-100`.

## 3. Настройка

Скопируйте `.env.example` в файл `.env`:

```env
BOT_TOKEN=токен_из_BotFather
STAFF_CHAT_ID=-1001234567890
ADMIN_IDS=123456789,987654321
DB_PATH=apd_bot.db
```

`ADMIN_IDS` содержит Telegram ID руководителей через запятую.

Если оставить `ADMIN_IDS` пустым, кнопки решений сможет нажимать любой участник закрытого чата. Лучше обязательно указать ID руководителей.

## 4. Запуск на Windows

Установите Python 3.11 или новее.

Откройте командную строку в папке проекта:

```bat
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python bot.py
```

## 5. Запуск на Linux

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python bot.py
```

## Команды

- `/start` открывает главное меню.
- `/cancel` отменяет заполнение анкеты.

## Хранение данных

Заявки сохраняются в SQLite-файле `apd_bot.db`.

Токен хранится только в `.env`. Файл `.env` нельзя публиковать или загружать в открытый репозиторий.
